# Mini Social Media Platform

A mini social media web application built using Django, HTML, CSS, and SQLite.

## Features

* User Registration and Login
* Create and View Posts
* Comment on Posts
* Like Posts
* Follow Other Users
* User Profiles
* SQLite Database Integration

## Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite

## How to Run

1. Clone the repository
2. Open the project folder
3. Install Django:
   pip install django
4. Run migrations:
   python manage.py migrate
5. Start the server:
   python manage.py runserver
6. Open:
   http://127.0.0.1:8000/

## Project Structure

* socialapp/
* socialmedia/
* templates/
* db.sqlite3
* manage.py
